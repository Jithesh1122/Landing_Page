# PulseFlow Landing Page

A modern React landing page for **PulseFlow**, designed to showcase product value through a focused hero section and feature highlights.

## Tech Stack

- React 18
- Vite 5
- CSS (custom styles)

## Features

- Product-focused hero section with clear call-to-action
- Smooth scroll from CTA to feature section
- Reusable feature card layout
- Responsive UI structure for desktop and mobile

## Project Structure

```text
src/
  components/
    FeatureCard.jsx
    FeaturesSection.jsx
    Hero.jsx
    LivePreview.jsx
  App.jsx
  main.jsx
  styles.css
```

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Notes

This project is intended as a clean starting point for product landing pages and can be extended with sections such as testimonials, pricing, and FAQ.
